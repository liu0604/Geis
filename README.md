## ![安装方式](https://raw.githubusercontent.com/GalAster/Deus/master/Resources/ico/board-game-map.png) Install

#### 手动安装

- 点击release下载最新的安装包

  - release 版本不带说明文档

- 解压到任意`$Path`路径下

  - 如果选择手动安装那么升级也要手动升级

#### 自动安装

- 使用``BTools` ``的部署功能
